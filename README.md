# Hospital Appointment & Triage System

This repository contains the **complete implementation** of a hospital appointment
and triage management system.

### Features
- Circular Queue for routine appointment handling
- Min-Heap emergency triage (severity-based)
- Doctor schedule via singly linked lists
- Patient storage using hash tables
- Undo stack for reversible operations
- Reporting (per-doctor, served vs pending, top frequent patients)
- Fully runnable Python module

### Running the System
```bash
python3 src/hospital_system.py
```
